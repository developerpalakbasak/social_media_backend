# Social Backend

This is the backend for a social media application built with **Node.js**, **Express.js**, **MongoDB**, and **Socket.IO**. It supports features like user authentication, real-time messaging, file uploads, and email notifications.

## Features

- User authentication with JWT
- Password hashing with bcryptjs
- File uploads using multer
- Cloud storage integration with Cloudinary
- Email notifications via Nodemailer
- Real-time communication with Socket.IO
- Logging with Morgan
- CORS support for cross-origin requests
- Environment variable management with dotenv

## Tech Stack

- **Node.js**  
- **Express.js**  
- **MongoDB & Mongoose**  
- **Socket.IO**  
- **Cloudinary**  
- **Nodemailer**  
- **bcryptjs**  
- **dotenv**  
- **cors**  
- **cookie-parser**  
- **multer**  
- **morgan**
